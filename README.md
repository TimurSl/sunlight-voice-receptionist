# How to run
- Setup .env with these keys:
```
BOT_TOKEN=
MODERATOR_ROLE_ID=
DISCORD_VOICE_ROOMS_CHANNEL_ID=
```
- `pip install -r requirements.txt`
- `python3 main.py`
